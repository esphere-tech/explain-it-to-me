# Security Policy

## Supported Versions

Only the latest release receives security updates.

| Version | Supported |
| ------- | --------- |
| 2.3.x   | ✅         |
| < 2.3   | ❌         |

Current version: **2.3.2**

---

## Reporting a Vulnerability

If you find a security issue, please report it by opening a **GitHub Issue** in the repository.

When submitting an issue, include:

* A clear description of the vulnerability
* Steps to reproduce (if possible)

We review security reports as quickly as possible and will follow up in the issue thread.
